The Knapsack problem

Description

The famous knapsack problem. You are packing for a vacation on the sea side and you are going to carry only one bag with capacity S. You also have N items that you might want to take with you to the sea side. Unfurtunately you can not fit all of them in the knapsack so you will have to choose. For each item you are given its size and its value. You want to maximize the total value of all the items you are going to bring. What is the maximum total value?

Input

On the first line you are given S and N. N lines follow with two integers on each line describing on of your items. The first number is the size of the item and the next is the value of the item

Output

You should output a single integer on one like - the total maximum value from the best choice of items for your trip.

Constraints

  1. 1 <= S <=2,000
  2. 1 <= N <=2,000
  3. Time limit: 0.240s
  4. Memory limit: 1,536MB

Example

Input Output
4 5
1 8
2 4
3 0
2 5
2 3 		13
